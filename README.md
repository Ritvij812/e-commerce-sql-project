# E-commerce Database Analysis using SQL

## Overview

This project focuses on designing and analyzing an e-commerce database using SQL. The project includes relational database schema creation, transactional data analysis, and extraction of business insights from customer and sales data.

## Features

* Relational database schema design
* SQL JOIN operations across multiple tables
* Revenue and sales analysis
* Customer spending behavior analysis
* Product performance tracking
* Cancelled order trend analysis

## Database Tables

* Customers
* Orders
* Products
* Payments

## Technologies Used

* SQL
* Relational Database Concepts
* Aggregate Functions
* Data Filtering and Sorting

## SQL Concepts Applied

* JOIN Queries
* GROUP BY
* ORDER BY
* Aggregate Functions
* Conditional Queries
* Data Filtering

## Key Insights Generated

* Total revenue calculation
* Identification of top-selling products
* Customer purchase behavior analysis
* Cancelled order trends
* Product-wise performance evaluation

## Learning Outcomes

* Improved SQL query writing skills
* Learned relational database design
* Developed data analysis and business insight generation skills
* Enhanced understanding of transactional database systems
